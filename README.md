## [AlcaláSuena](http://www.alcalasuena.es) - Android app

Software to manage music festivals:

- Events list
- Choose favourites
- Genre filter
- Bands info
- Venues info
- Geolocalization
- Push notifications
- Notices


Developed by [Triskel Apps](https://triskelapps.com) at [Ínsula Coworking](http://insulacoworking.es)
